## 1. Objective
Automate and validate the Login functionality using Playwright to ensure:
- Correct credentials allow successful authentication.
- Incorrect credentials are rejected with appropriate error messages.
- UI elements behave as expected.
- Application redirects to the correct pages post-login.

---

## 2. Scope
Automation includes:
- UI checks
- Functional checks
- Validation messages
- Navigation/redirect behavior
- API/network verification (optional)

Out of scope:
- Backend authentication logic
- Database validation
- Third-party login (Google, SSO, etc.)
